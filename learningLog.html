Znalazłem jakieś rozwiązanie któej było użytye nawet w naszym zadniu zagadce lecz nie był integracji z git na której mi zależy żeby łatwo przenosić kod miedzy tabletem i PC. Możliwe że opcja jeszcze dla mnie bedzie visual studio code online do czasu aż nie kupie sobie laptopa.
